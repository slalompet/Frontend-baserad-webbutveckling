Advantages of Using Sass

  • Better organization and structure
  • Effortless to keep it consistent by using variables
  • Developers can adjust the color by using color function efficiently
  • Simple and clean codes, the systax is logical and easy to read in order to apply changes
  • Powerful tool by using nesting and mixins and it allows developers to reuse codes

Disadvantages of Using Sass

  • Browsers cannot read SCSS codes, and they need to be conpiled
  • So far I don't find it saves time due to the learning curve
  • Trouble shooting, it can be challenging to identify the issue when it's complex
  • For some reasons, I need to run npm start each time I encounter errors

Conclusion

  • I'm using the same repo from moment 1 and moment 2
  • It's fun to learn a CSS preprocessor but it takes times to know the rules and limitations
  • Not sure how I would use those flow controls for my own project but it definitely helps reduce the duplication of the code
