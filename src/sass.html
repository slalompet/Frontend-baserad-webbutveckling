Frontend-baserad webbutveckling, Moment 3

Here is how I do this assignment

Set up the project

  • Create a repo on Github
  • Clone the repo to my local device
  • Start with npm init -y in the terminal
  • Remove main and script in the package.json file
  • Install parcel
  • Added gitignore for the files not going to commit
  • Set up package script and created src folder

During development

  • Created folders and subpages to set up the basic structure for the website
  • Start the next.js server by using npm run start command to get the live update of the website
  • Added menu bar and 5 images according to the requirement
  • Added js digital clock code from Digital Clock in JavaScript

Publishing

  • Push all the commits to the remote repo
  • Finish the project and create a build directory by using npm run build command
  • Published the website by using main branch on GitHub Pages

Tools

  • Github is used for version control and publishing the website
  • Parcel is used for removing unused codes and minifying JavaScript, CSS and HTML.
    Besides, Parcel optimizes the images and help with the browser compatibility to ensure the backward compatibility for users from what I understand.
